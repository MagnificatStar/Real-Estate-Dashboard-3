# Real-Estate-Dashboard
This Real Estate Dashboard provides key metrics for a Real Estate Agency or a Property Management Company
## Dataset used 

- <a href = "https://github.com/MagnificatStar/Real-Estate-Dashboard-3/blob/main/real_estate.xlsx" >Dataset </a>
## Questions (KPIs)
-	What is the average list price?
-	How many properties currently have active listings?
-	Compare property distribution percentages across Residential, Rental, and Commercial categories, and display the count of active, sold, and pending properties using a donut chart.
-	Which city has the most properties listed? 
-	What is the price trend by city using a clustered column chart? 
-	Compare agent performance by showing the average commission percentage, average properties sold, and close rate percentage.
